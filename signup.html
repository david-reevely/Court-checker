Ontario Courts Monitor

This tool runs a daily search of new civil court case filings in Toronto. The Ontario government is making its court records more available online; it has started with Toronto cases, which covers a lot of useful ground. The tool searches only civil filings in Toronto, and only reports on NEW CASES, not new filings in existing cases. Particular filings, such as statements of claim, are not available online (though the website has features that make it look as though they eventually might be). You'll have to request them by either going to the courthouse or by phoning or emailing the clerk.

This thing is pretty ugly on the back end. Filling out this form generates a configuration file to send to David, which goes into the tool's maw for use the next time it runs. The first time, you'll likely get a large email showing all the open cases that match your search terms. Future runs will send you only newly filed ones. An elegant and easy way to edit the list will come in the future.
